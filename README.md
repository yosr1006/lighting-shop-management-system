# lighting-shop-management-system
A relational database management system for a lighting and chandelier shop, 
built with MySQL, following database design principles, and integrated 
with a Flask + HTML/CSS/JS web interface.
## Features
- Product, customer, and supplier management
- Sales invoices with automatic discounts (loyalty / bulk)
- Purchase invoices with stock tracking
- Product returns with a 14-day return policy
- Live search across tables
## Tech Stack
- Python (Flask)
- MySQL
- HTML / CSS / JavaScript
## Setup
1. Clone the repo
2. Install dependencies: `pip install flask mysql-connector-python`
3. Set environment variables for DB_HOST, DB_NAME, DB_USER, DB_PASSWORD
4. Run: `python app.py`
